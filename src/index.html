Face Detection App

A single page application that detects faces in image URLs entered by registered users. Using Clarifai's machine learning API, the locations of the faces in the image are obtained from the response. Bounding boxes are drawn around the faces after further calcluations. User profiles are updated upon each image entry.

The app implements features such as: user registration, user authentication, RESTful API, frontend & backend routing.

  • What's under the hood?
  • React.js
  • Node.js
  • Express.js
  • PostgreSQL
  • Heroku

LinkedIn Profile Scraper

A Google Chrome extension that extracts information from open LinkedIn profile tabs. Data from the profiles is read by the extension and sent to a local Node.js sever. The server reads the data, represented as an array of objects, and SheetJS to export the data in a table format to an Excel workbook.

  • What's under the hood?
  • VanillaJS
  • Node.js
  • Express.js
  • SheetJS

Reacher - AI Robot Teacher

Reacher is an AI & computer-vision based teaching assistant robot that primarily consists of 5 subsystems: (a) an autonomous navigation subsystem for exam proctoring, (b) a trained LSTM model for cheating detection, (c) a contactless CV-based exam paper scanning subsystem, (d) a human-robot voice interaction subsystem, (e) and an academic content personalization subsystem.

  • What's under the hood?
  • LSTM-based CNN in MATLAB
  • ROS (SLAM & Dijkstra's algorithm)
  • Python's Porcupine engine
  • Google's DialogflowCX for NLP
  • WebSockets
  • Facial recognition
  • Recommendation engine
